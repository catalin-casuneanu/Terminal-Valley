# Terminal Valley

Turn based rpg game.

## Folder Structure

1. Backend
	Vanilla Node.js backend for terminal_valley. 
	Contains server and game engine.
2. Terminal_valley
    Nuxt 3 frontend.
3. Sprite_Game
    Old school vanilla js in-browser game. 
    Made to research how sprite animations work.
